# hexo-pages

[Hexo](https://hexo.io/) plugin to generate a static pages.


## Install

``` bash
$ npm install hexo-pages --save
```


## Usage

Create a dir named `pages` and create your pages here.

Example:

```bash
pages
├── page-1
    ├── index.html
```

Page url will be: http//my-blog.com/page-1
